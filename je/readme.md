# dependencies

you need `vipe` from `moreutils` package and `fzf`

# how-to

add this line in your in your `~/.zshrc`

  . ~/path/to/je.zsh

or just

  . je.zsh

if `je.zsh` is in the path.

or

  uze je

if you're using `uze`.
